WHY: The Business Challenge

🎯 Primary Business Question

"How can an online store understand their customers better to increase sales and customer loyalty?"

📈 Business Impact & Background

In today's competitive e-commerce landscape, treating all customers uniformly is inefficient and costly. This project addresses the critical need to move beyond a "one-size-fits-all" approach.

WHAT: Objectives & Deliverables

🎯 Primary Objective

To segment the e-commerce customer base into distinct, actionable groups based on shopping behavior to enable hyper-personalized marketing and drive business growth.

📋 Key Deliverables

  • Customer Segment Profiles - Detailed characteristics and RFM analysis
  • Interactive Tableau Dashboard - Real-time monitoring and decision support
  • Business Recommendations - Actionable strategies per segment
  • Predictive Churn Model - Early warning system for at-risk customers
  • A/B Testing Framework - Statistical validation methodology
  • Complete GitHub Repository - Documented, reproducible analysis

HOW: Project Scope

✅ IN SCOPE

  • Customer segmentation using RFM analysis
  • K-Means clustering and validation
  • Churn prediction modeling
  • Interactive dashboard creation
  • Statistical hypothesis testing
  • A/B testing framework design
  • Business recommendations

🚀 Minimum Viable Product (MVP)

  • 3-5 core customer segments identified
  • Basic RFM analysis and profiling
  • Simple Tableau dashboard with key metrics
  • Top 3 business recommendations

📊 Data Catalog

Data Source: Kaggle "Transactional E-commerce" Dataset by Bytadit

Multi-table relational dataset with millions of records over time

👥 Customer Table

  • customer_id (Primary Key)
  • gender, device_type
  • home_location_lat/long
  • first_join_date

🛍️ Product Table

  • id (product_id)
  • masterCategory, subCategory
  • articleType, baseColour
  • season, productDisplayName

💳 Transaction Table

  • created_at, customer_id
  • booking_id, session_id
  • product_metadata (JSON)
  • payment_method, total_amount

📱 Click Stream Table

  • session_id, event_name
  • event_time, traffic_source
  • event_metadata (JSON)
  • App usage activities
